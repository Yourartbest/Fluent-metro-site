# Fluent Design / Metro Website

A professionally designed website showcasing Microsoft's Fluent Design and Metro design language principles.

## About This Project

This website explores Fluent Design (formerly Metro), Microsoft's typography-first design language that revolutionized digital interfaces. Built following authentic Fluent/Metro principles from the comprehensive style guide.

## Design Characteristics

### Typography
- **Segoe UI** font family (with Inter fallback)
- **Dramatically large headings**: Hero text at 96px, H1 at 56px
- **Bold weights**: Semibold (600) for hierarchy
- **Sentence case**: Readable, not ALL CAPS
- **Left-aligned**: Strong flush-left alignment

### Color Palette
- **Microsoft Blue**: #0078D7 (primary)
- **Metro Colors**: Magenta (#D80073), Teal (#00ABA9), Orange (#FA6800), Lime (#A4C400)
- **High contrast**: White text on colored backgrounds
- **Neutral base**: White and light gray backgrounds

### Layout
- **Tile-based grid**: Signature Metro aesthetic
- **Varied proportions**: Different tile sizes create visual interest
- **Strict alignment**: Everything snaps to grid
- **Responsive**: 12-column grid system

### UI Elements
- **Rounded corners**: 4-8px border radius (Fluent evolution)
- **Subtle shadows**: 0 2px 8px rgba(0,0,0,0.1)
- **Fast animations**: 200-300ms with ease-out timing
- **Card-based**: Modern Fluent approach

### Core Principles Implemented
1. **Content before chrome** - Information prioritized over decoration
2. **Typography dominance** - Text as primary design element
3. **Bold color blocks** - Pure, saturated Metro palette
4. **Grid alignment** - Strict structural order
5. **Information density** - Efficient use of screen space
6. **Authentically digital** - No skeuomorphism
