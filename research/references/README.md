# Reference Images Folder

This folder is intended to contain reference images used during the design research phase.

## Recommended Reference Images to Add:

### Historical Metro Examples
- Windows Phone 7 Start screen
- Windows 8 Start screen with Live Tiles
- Original Metro design guidelines screenshots

### Contemporary Fluent Examples
- Windows 11 Start menu
- Microsoft Teams interface
- Azure Portal dashboard
- Microsoft 365 apps

### Typography Examples
- Segoe UI font specimens
- Metro typography hierarchy examples
- Windows 11 typography scales

### Color Palette References
- Metro color swatches
- Windows Phone color themes
- Fluent Design color system

## How to Use
1. Save reference images to this folder
2. Name them descriptively (e.g., `windows-phone-7-start-screen.png`)
3. Document sources in `../references.md`
4. Use for comparison during design iterations

Images are not included in this repository to respect copyright. Use official Microsoft design resources and documentation for reference.
