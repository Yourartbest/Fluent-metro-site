## Quick Generation Tips:

### For Midjourney:
- Add `--ar [ratio]` for aspect ratio
- Add `--style raw` for less stylized, more literal results
- Add `--v 6` to use latest version
- Use `--no shadows, gradients` to enforce flat Metro style

### For DALL-E 3:
- Be explicit about "no shadows, completely flat design"
- Specify "Microsoft Metro style" or "Windows Phone 7 aesthetic"
- Request "high contrast, bold colors"

### For Stable Diffusion:
- Use terms like "UI screenshot", "interface design", "flat design"
- Add "professional, clean, minimalist" for Metro aesthetic
- Negative prompts: "shadows, gradients, textures, skeuomorphism, 3D"
